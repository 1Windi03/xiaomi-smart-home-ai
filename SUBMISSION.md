## 📋 Project Overview

AI-powered smart home control dashboard that demonstrates the integration of:
- **Artificial Intelligence** - Natural language processing for device control
- **IoT Simulation** - Smart home device management
- **Real-time Communication** - WebSocket-based live updates
- **Modern Web Technologies** - Responsive, interactive UI

---

## ✨ Key Features

### 1. Natural Language Control
Users can control devices using plain English commands:
- "turn on living room light"
- "set bedroom light to 50%"
- "lock all doors"
- "turn off everything"

### 2. Voice Commands
Hands-free control using Web Speech API - click the microphone and speak your command.

### 3. Real-time Dashboard
- Live device status updates
- Energy consumption monitoring
- Command history tracking
- Security status overview

### 4. AI-Powered Intelligence
- OpenAI GPT-3.5 integration for command parsing
- Fallback demo mode for offline operation
- Context-aware device control

### 5. Device Types Supported
- 💡 Lights (brightness control)
- ❄️ Air Conditioners (temperature control)
- 🌀 Fans (speed control)
- 🔒 Smart Locks
- 🚪 Garage Doors
- 📹 Security Cameras

---

## 🛠️ Technical Stack

**Backend:**
- Node.js + Express
- WebSocket (ws) for real-time communication
- OpenAI API integration

**Frontend:**
- Vanilla JavaScript (no framework dependencies)
- HTML5 + CSS3
- Web Speech API for voice control
- Responsive design (mobile-friendly)

**Architecture:**
- Event-driven WebSocket communication
- Simulated IoT device state management
- Energy usage tracking
- Command history logging

## 🚀 Installation & Usage

```bash
# Clone repository
git clone https://github.com/1Windi03/xiaomi-smart-home-ai.git
cd xiaomi-smart-home-ai

# Install dependencies
npm install

# Configure (optional - works without OpenAI key)
cp .env.example .env
# Edit .env to add OPENAI_API_KEY if desired

# Start server
npm start

# Access dashboard
# Open http://localhost:3000 in browser
```
