AI Smart Home Dashboard - Natural language control for smart home devices with voice commands.

Built with AI + IoT integration featuring:
• Natural Language Processing - Control devices with plain English ("turn on living room light")
• Voice Commands - Hands-free control using Web Speech API
• Real-time Dashboard - WebSocket-based live updates across all clients

• AI-Powered - OpenAI GPT-3.5 integration + offline demo mode
